# Todo-API-Golang-
Todo list REST API with golang-echo and mysql 

Feature: 
1. Add Todo
2. Delete Todo
3. Edit Todo
4. Display Particular Todo
5. Display All Todo
6. Mark as Completed 
7. Login User
8. SignIn User
9. Logout User
